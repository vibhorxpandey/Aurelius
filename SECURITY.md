# Security

If you discover a security vulnerability, please do not open a public issue. Instead:

1. Email the maintainers at `@vibhorxpandey` via GitHub or use GitHub's private security advisory flow.
2. Provide a short description, steps to reproduce, and an impact assessment. Attach PoC if possible.

We will acknowledge receipt and aim to respond within 72 hours.

Recommended process for maintainers
- Triaging via GitHub Security Advisories.
- Backport fixes to any supported release branches.
- Publish an advisory and release a patched version.
